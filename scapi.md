# API 文档

## /skey/get
### 描述
获取密钥列表。
### 请求参数
- `authkey`：授权密钥（必填）
- `page`：页码（可选，默认为1）
- `pageSize`：每页大小（可选，默认为10）
### 响应
- 成功：返回密钥列表的 JSON 数据
- 失败：返回错误信息

## /skey/del
### 描述
删除指定的密钥。
### 请求参数
- `authkey`：授权密钥（必填）
- `id`：密钥 ID（必填）
### 响应
- 成功：返回操作结果的 JSON 数据
- 失败：返回错误信息

## /skey/add
### 描述
添加新的密钥。
### 请求参数
- `username`：用户名（必填）
- `type`：密钥类型（必填）
- `content`：密钥内容（必填）
- `qq`：QQ号（可选，如果类型为1，则必填）
### 响应
- 成功：返回操作结果的 JSON 数据
- 失败：返回错误信息
