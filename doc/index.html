API Documentation

发空间说说 (PostQQZoneMessage)

URL: /app/zone/post

Method: GET

Parameters:

  • qq: QQ号码
  • text: 说说内容
  • key: ClientKey
  • ptlang: ClientKey Type (QQ:2052 NT:1033) (默认: 2052)

设置/取消指定好友为特别关心 (SetCare)

URL: /app/zone/care

Method: GET

Parameters:

  • qq: QQ号码
  • key: ClientKey
  • tqq: 特别关心的QQ号码
  • set: 设置为特别关心 (true) 或取消 (false)
  • ptlang: ClientKey Type (QQ:2052 NT:1033) (默认: 2052)

修改信息 (ChangeUserInfo)

URL: /app/user/changeinfo

Method: GET

Parameters:

  • qq: QQ号码
  • key: ClientKey
  • newName: 新昵称
  • ptlang: ClientKey Type (QQ:2052 NT:1033) (默认: 2052)
  • company: 公司信息 (默认: 奶龙园区祝你天天开心)
  • address: 地址信息 (默认: 奶龙园区祝你天天开心)

服务器内接口调用文档


注册 (Register)

URL: /auth/reg

Method: GET

Parameters:

  • username: 用户名
  • password: 密码
  • email: 邮箱
  • qq: QQ号码
  • userkey: 用户密钥

登录 (Login)

URL: /auth/login

Method: GET

Parameters:

  • username: 用户名
  • password: 密码

添加用户密钥 (AddUserKey)

URL: /auth/addkey

Method: GET

Parameters:

  • userKey: 用户密钥 (自动生成)
  • level: 等级参数

获取用户信息 (Info)

URL: /auth/info

Method: GET

Parameters:

  • authkey: 验证密钥

添加 SKEY (Add SKEY)

URL: /skey/add

Method: GET

Parameters:

  • authkey: 验证密钥

添加 API SKEY (Add API SKEY)

URL: /skey/addapi

Method: GET

Parameters:

  • username: 用户名
  • content: base64编码后的RC4 (点号需要去除)

获取 SKEY (Get SKEY)

URL: /skey/get

Method: GET

Parameters:

  • authkey: 验证密钥
  • username: 用户名

删除 SKEY (Del SKEY)

URL: /skey/del

Method: GET

Parameters:

  • authkey: 验证密钥
  • username: 用户名
  • skeyID: SKEY ID

构建 (Build)

URL: /skey/build

Method: GET

Parameters:

  • authkey: 验证密钥
  • username: 用户名
  • mail: 邮箱
  • keymode: 密钥模式
  • injectname: (可选) 注入名称 (仅在密钥模式为其他时需要)
