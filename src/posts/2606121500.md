## Libra-Nextgen 是一款专为现代企业级网络安全红蓝对抗设计的高性能、高隐蔽性 C2（Command and Control）框架。

项目采用经典的三层分离架构，提供从终端植入、远程控制到数据窃取的全方位后渗透能力，支持多操作员实时协同作战。

**GitHub：** [github.com/SmaZone2020/Libra-Nextgen](https://github.com/SmaZone2020/Libra-Nextgen)

---

## 技术架构

```
┌──────────────────┐    WebSocket / REST     ┌──────────────────┐
│  Libra-Console   │ ◄─────────────────────► │  Libra-Server    │
│  (React Web UI)  │   JWT 认证 + RBAC       │  (ASP.NET Core)  │
│  多人协同控制台   │   实时状态同步           │  MongoDB 持久化   │
└──────────────────┘                         └────────┬─────────┘
                                                      │
                                         AES-256-GCM  │  WebSocket
                                         + RSA 协商   │  / HTTP(S)
                                                      │
                                           ┌──────────┴──────────┐
                                           │  Libra-Agent         │
                                           │  (.NET Native AOT)   │
                                           │  内存驻留 模块化载荷  │
                                           └─────────────────────┘
```

| 层级 | 技术栈 | 职责 |
|------|--------|------|
| **Libra-Agent** | .NET 10 Native AOT | 轻量级植入体，内存驻留，按需加载后渗透模块 |
| **Libra-Server** | ASP.NET Core WebAPI + WebSocket | 流量接入、任务调度、数据持久化 |
| **Libra-Console** | React + Vite + HeroUI + TailwindCSS | 多人协同 Web 操作平台 |

操作员登录后的全局态势总览。实时展示 Agent 在线统计、任务分布、流量趋势，结合 GeoIP 数据在交互式地图上标记所有设备的物理位置。

## 设备管理

管理所有已上线 Agent，提供深度硬件指纹采集 —— CPU 型号与核心数、GPU 信息、磁盘序列号、主板 BIOS 版本、电池状态等。支持按在线状态筛选，一键下发凭据抓取命令。

![设备信息](/images/85a958b9d1d24a21a306610718702eea.png)

## 远程终端

摒弃传统单行命令执行模式，提供完整交互式伪终端（PTY）。Linux 端支持 vim、top、nano 等交互式命令，Tab 补全与历史回溯；Windows 端提供完整执行上下文的 PowerShell 会话。基于 xterm.js 渲染，WebSocket 毫秒级同步。

![终端](/images/109934d8daca48c597dc1edbf808c878.png)

## 屏幕监控

静默监控目标桌面画面。支持 1–15 FPS 多档帧率与多档分辨率，采用图像差分压缩算法仅传输变化像素区块，大幅降低带宽消耗。通过底层 DirectShow / Media Foundation 接口捕获，规避部分 EDR 的 Hook 检测。

![屏幕监控](/images/a079fab91b104236bcb363d366710c64.png)

## 摄像头与麦克风

针对配备外设的终端，支持静默抓拍、低帧率视频录制与实时音频流传输。通过底层接口直接读取数据流，自动检测可用设备列表，支持 5–30 FPS 帧率调节。

![媒体监控](/images/b194769116f44d3ab449375e92c612f0.png)

## 文件管理

完整的远程文件系统管理：驱动器浏览、目录导航、上传下载、重命名、移动、复制、删除、压缩解压。大文件采用分块传输与断点续传机制。支持反取证时间戳伪造（Timestomping），抹除操作痕迹。

## 软件数据窃取

### 浏览器凭据

支持 Chrome（Chromium 系）、Edge、Firefox 的密码、Cookie、浏览历史提取。Chromium 系通过 DPAPI 解密 AES Master Key 后 AES-256-GCM 解密；Firefox 通过 key4.db 提取 Master Key 后 PBES2 + 3DES/AES-GCM 解密。

### AI 密钥扫描

检测 6 款主流 AI 编码工具本地存储的 API 密钥 —— Claude Code、OpenCode、CodeX、Gemini、OpenClaw、HermesAgent。默认脱敏显示（前4位+后4位），支持一键切换明文。

![AI密钥扫描](/images/b3409da180234e65a015dfb1be89afcd.png)

### 微信 & QQ

自动定位数据文件夹，枚举所有登录过的账号（wxid / QQ号）。QQ 支持通过服务端代理调用 QQ 空间 API 获取头像与昵称，规避浏览器跨域限制。

## 反向代理

利用 Agent 的网络环境代理浏览网页，实现内网穿透。支持多标签页、URL 导航、前进后退。所有请求以 Agent 的 IP 和网络环境发出，可访问内网 Web 应用或绕过 IP 限制。

![反向代理](/images/b558683f8def41468c01e5b59abd733b.png)

---

## 分布式压力测试

分布式多点 DDoS 模拟攻击平台，支持从在线 Agent 中选择攻击节点，对目标 IP/域名 + 端口发起攻击。涵盖四层（SYN Flood、UDP Flood、ICMP Flood、反射放大）与七层（HTTP Flood、Slowloris、TCP 连接耗尽、协议畸形包）共 8 种攻击方式。实时流量图表（ECharts 曲线）、状态面板与历史记录。

![压力测试](/images/24998d89855644a9a2360be105fcb69d.png)

## 系统信息收集

### 网络侦查

网卡信息、公网 IP、ISP、ASN、经纬度、Wi-Fi 密码提取、附近 Wi-Fi 热点扫描（SSID + BSSID + 信号强度）、ARP/NetBIOS 局域网设备发现。

### 本地账户与凭据

枚举系统本地用户账户（账户名、SID、管理员权限、密码策略），支持一键内存凭据抓取（Mimikatz），提取 LSASS 进程中的明文密码与哈希值。

![本地账户](/images/0739c802153d4e2d9e86b1bb1d243211.png)

### 窗口管理

枚举所有可见窗口（句柄、标题、进程名），支持置顶、置底、最小化、最大化及远程修改窗口标题。

![窗口管理](/images/585ffd74341b4b748f08e86cc0843ab8.png)

自助式 Agent 生成工具，操作员可自定义平台（Windows/Linux）、连接配置、元数据伪装、文件图标。支持 Obfuscar 混淆、未使用代码裁剪、注入垃圾数据（改变文件哈希特征）。所有构建历史保存并可追溯下载。

## 安全机制

- **全链路加密**：Agent-Server 通信强制 AES-256-GCM 加密，密钥通过 RSA 非对称协商，每 Agent 独立会话密钥
- **流量伪装**：Malleable C2 Profile 动态修改 HTTP 请求特征，将 C2 通信伪装为正常业务 API
- **JWT + RBAC**：控制台 RS256 非对称签名认证，区分 Operator / Admin 权限角色
- **强制审计**：所有指令写入 MongoDB AuditLogs，前端不提供删除权限
- **反沙盒/反虚拟机**：检测 VMware/VirtualBox/Hyper-V 特征，校验 CPU 核心数、内存大小、鼠标轨迹等环境探针
- **一键自毁**：赛事结束后向所有 Agent 下发清理指令，自动撤销持久化项，释放内存并终止进程
